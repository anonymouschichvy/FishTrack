How Onboard Image Classification Works

What does this monitor do? It displays results from underwater camera captures that are automatically classified by buoy nodes. To conserve radio bandwidth, the full image is saved locally on the buoy's SD card, while a compressed JSON telemetry packet (containing species labels, bounding boxes, and confidence thresholds) is transmitted back to the Base Station via LoRa.

Why is it important? Automated marine tracking allows continuous biological surveys without requiring physical diving or high-power cellular transmitters. Biologists can inspect species frequency curves and verify detection statistics in real-time.

Fish Monitor Docs

What is YOLO? YOLO (You Only Look Once) is a fast, state-of-the-art neural network architecture used for object detection. The buoy uses a pruned YOLO model tuned specifically to identify marine species in low-light and turbid waters.

Verification Guide

How to verify classifications: Each photo log entry displays a confidence level percentage (e.g. 88%). If the confidence level is low (below 75%), check for turbid/muddy water conditions or camera lens fouling.
