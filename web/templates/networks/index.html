Understanding Self-Healing Ad-Hoc Networks

What does this visualizer do? It maps the spatial topology and hop vectors of the buoy sensor network. Node connections represent active LoRa RF links. The Base Station acts as the network sink node (the ultimate receiver of all data).

What is "Self-Healing"? Marine environments are unstable; waves and obstacles can break line-of-sight. If a node cannot contact the Base Station directly (Hops = 1), it dynamically routes its data through an intermediate neighbor (Hops > 1). The network automatically reorganizes pathways to ensure telemetry packet delivery.

Network Docs

Mesh Topology: The Base Station occupies Node ID 0. Nodes route packets dynamically through paths with the highest RSSI/SNR signal values to avoid drops.

Signal Criteria

RSSI Values:
• -30 to -70 dBm: Excellent
• -70 to -90 dBm: Good
• -90 to -110 dBm: Poor
• -110+ dBm: Disconnected

Glossary

Hop Count
The number of intermediate relay transmitters a packet travels through from source buoy to base station.
SNR (Signal-to-Noise Ratio)
The difference in dB between the received signal level and the background ambient radio noise floor.
