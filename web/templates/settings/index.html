Optimizing Network Radio Intervals

What do these settings control? These options configure parameters that govern mesh network coordination. Discovery Interval defines how often nodes ping each other to build neighbour databases. Expected Buoy Nodes lists all nodes active in your fleet.

Why is interval timing critical? In LoRa, sending packets frequently wastes battery and creates congestion. Setting the discovery interval too low (e.g. 10s) causes collisions; setting it too high (e.g. 600s) makes status updates sluggish. Operators must balance responsiveness against battery life.

Glossary

Discovery Ping
An RF beacon broadcast containing the sender's codename and hops configuration, used by receivers to map neighbor nodes.
FLEET IDs
The list of allowed node identifiers that the base station will accept and track.
