How Automated Job Scheduling Works

What does this scheduler do? The Job Scheduler manages automated command execution. Operators can configure tasks (such as a sonar calibration or a battery check) to run once, at a set interval, or on a recurring daily/weekly cron-like pattern.

Why run scheduled jobs? Manual monitoring of multiple buoy nodes is labor-intensive. Scheduled jobs automate routine tasks, such as triggering a 10-minute high-resolution sonar profile at midnight, or reporting battery capacities daily, ensuring consistent datasets without human intervention.

Scheduler Docs

Frequency Options: Hourly, Daily, and Weekly run relative to the base station local clock. Interval schedules run based on a delta time stamp offset.

Duty Cycle Limit

LoRa Constraint: Avoid scheduling heavy tasks on multiple buoys at the same minute to prevent radio frequency collision.

Glossary

One-shot Job
A scheduled task configured to execute exactly once and then self-delete from the scheduler queue.
Interval Job
A task scheduled to repeat dynamically after a defined number of elapsed minutes.
