Welcome to the FishTrack Base Station

This dashboard monitors a mesh network of buoy sensors positioned in the ocean. Let's look at what the key telemetry monitors track:

Glossary of Technical Terms

LoRa (Long Range)
A spread-spectrum modulation technique derived from chirp spread spectrum (CSS) technology, designed for low-power, long-distance telemetry.
RSSI (Received Signal Strength Indicator)
A measurement of the power present in a received radio signal. Closer to 0 dBm is stronger; marine links usually operate between -30 dBm and -110 dBm.
SNR (Signal-to-Noise Ratio)
The ratio of signal power to noise power. Positive values indicate a strong signal, while negative values indicate the signal is below the noise floor.
Buoy Mesh Node
An autonomous sensor buoy equipped with solar panels, battery charger, GPS receiver, sonar scanner, camera, and a LoRa radio to relay messages back to the Base Station.

System Operations Manual

The FishTrack Base Station coordinates marine mesh telemetry from multiple remote buoys. The key modules and features of the system include:

1. Dashboard & Connected Buoys

Provides a summary of active nodes, message counts, and USB connection health to the radio hardware. The Live System Feed aggregates real-time JSON packets, ping events, and warnings.

2. Interactive Routing Map

Visualizes spatial node layouts and telemetry flows using a D3 force-directed link model. Lines are color-coded based on signal strength. Hovering over a node displays battery levels, hops, and peer lists.

3. Command Center

Manages preset script dispatches. Presets are highly optimized binary indicators pre-compiled on buoy nodes to execute tasks (like camera captures or sonar sweeps) without overloading LoRa links.

4. Linux Subprocess Console

Allows operators to run shell commands on buoys. For safety, execution is limited to a 15-second timeout, and commands matching security filters (e.g., rm -rf, reboot loops) are blocked.

5. Job Scheduler

Automates periodic telemetry gathering (Hourly, Daily, Weekly, or specific intervals). Schedule timing should be staggered to avoid radio collisions (LoRa duty cycle limits).

6. Fish Detection Monitor

Displays counts and logs from computer vision object detection models (YOLOv8n) running on the buoy camera feed. Use confidence thresholds to identify false detections caused by turbid waters or biofouling.

7. Sonar Profiler Monitor

Controls and plots 200 kHz ultrasonic echograms. Measures acoustic reflections to estimate biomass thickness and underwater depth profiles up to 100 meters.

8. GPS Tracking & Geofencing

Tracks buoy drift using latitude and longitude coordinates. Fix quality levels indicate positional accuracy (3D, 2D, or No Fix). Drifts exceeding the geofence radius trigger alert highlights.

9. Battery Health & BMS

Monitors LiFePO4 cells. Shows individual cell voltages, balancing deviation, charger inputs (VBUS), and temperatures. Deviations above 150 mV highlight potential cell degradation.

10. Settings & Fleet Registry

Configures fleet authorization lists, serial port settings, and server endpoints. Unregistered nodes showing up in radio logs must be added to the allowed Fleet IDs list to begin tracking.

11. UI Controls & Operator Tools

Includes a dark mode theme selector, quick search command palette (Ctrl+K), guided onboarding tutorial wizard, and direct feedback submission modal for reporting observations.

Master Glossary of Terms

LoRa (Long Range)
A spread-spectrum modulation technique derived from chirp spread spectrum (CSS) technology, designed for low-power, long-distance radio telemetry.
RSSI (Received Signal Strength Indicator)
A measurement of radio power in received packets, measured in dBm. Closer to 0 dBm is stronger (-50 dBm is excellent, -110 dBm is near drop).
SNR (Signal-to-Noise Ratio)
The ratio of signal power to background noise power in decibels. Positive is a clear signal; negative means the signal is below the noise floor.
Sink Node
The master gateway receiver (Node 0, Base Station) where all multi-hop radio telemetry payloads terminate.
Hop Count
The number of relay jumps a radio packet takes through peer buoys to travel from its source node to the Base Station.
Mesh Routing
A self-healing network topology that redirects packets through intermediate neighboring nodes when a direct radio link is blocked.
ACK (Acknowledgement)
A small packet returned by a receiving buoy node to confirm it successfully demodulated and accepted a command transmission.
Duty Cycle
The percentage of time a radio transmitter is active on a channel, limited by hardware constraints and regulations to prevent collision.
Pico Serial Gateway
A Raspberry Pi Pico board running custom gateway firmware connected via USB to act as the RF-to-Base-Station bridge.
Discovery Beacon
An periodic radio broadcast sent by nodes containing their codename and neighbor link metrics, used to construct topology maps.
YOLO (You Only Look Once)
A fast, real-time object detection neural network running on the buoy to classify fish species from camera frames.
Confidence Level
A percentage representing the model's certainty. Scores under 75% indicate turbid water, poor lighting, or camera fouling.
Camera Viewfinder
The visual pane showing live or cached snapshots from the camera on a selected buoy node.
Bagre marinus
Scientific name of the Gafftopsail Catfish, commonly detected in regional marine survey areas.
Penaeus aztecus
Scientific name of the Brown Shrimp, serving as a key indicator class for benthic marine health.
Lutjanus campechanus
Scientific name of the Red Snapper, monitored around offshore rigs, reefs, and pipelines.
BMS (Battery Management System)
Electronic protection circuit tracking individual cell voltages and balancing battery cells during charging phases.
LiFePO4 Chemistry
Lithium Iron Phosphate cells. Chosen for buoy nodes due to thermal stability, long cycle life, and high safety factor in marine elements.
VBUS
Voltage at the charging bus input, representing power harvested from solar panel arrays before conversion.
Cell Balancing
The process of shunting charge to equalize individual cell voltages, keeping cell deviation below 150 mV.
C-Rate
A metric indicating battery charge/discharge speeds relative to total capacity (e.g., 0.5C rate).
Thermal Runaway
A dangerous reaction chain in batteries caused by excessive heat. Prevented by safety thresholds in the BMS telemetry.
GNSS
Global Navigation Satellite System, the general term for satellite constellations (GPS, GLONASS, Galileo, BeiDou).
3D Fix vs. 2D Fix
A 3D Fix locks 4+ satellites (horizontal coordinates + altitude); a 2D Fix locks 3 satellites (horizontal only); No Fix is searching.
Anchor Geofence
A virtual boundary radius centered on deployment coordinates. Exceeding this triggers drifting notifications.
NMEA-0183 Sentence
The standard ASCII text format (e.g., $GPRMC) sent by GPS hardware receivers to report location, time, and quality.
Drift Threshold
The maximum allowed distance (in meters) a buoy can move from its anchor point before being flagged as drifting.
Exit Code
The status integer returned by a completed subprocess. Code 0 indicates success; non-zero indicates an execution error.
Subprocess Shell
An isolated command executor session run in the background on the buoy's local Linux environment.
Waitress Server
A production-ready, multithreaded WSGI web server running the Python base station application.
Flask Framework
The lightweight python micro-framework hosting the REST API endpoints and rendering templates.
LocalStorage
Client-side browser database storage used to persist dark mode themes, cleared notifications, and layout preferences.
Command Blocklist
A security filter implemented in the buoy software that automatically rejects hazardous shell commands.
